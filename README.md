# bex 插件开发

## 环境要求

shell 环境使用 [MSYS2](https://www.msys2.org/)

编译工具链使用 GCC 11.3.0
```
gcc.exe (Rev1, Built by MSYS2 project) 11.3.0
GNU ld (GNU Binutils) 2.38
GNU Fortran (Rev1, Built by MSYS2 project) 11.3.0
```

> Note: 环境要求出自 "北太天元技术支持" QQ 群
